# 🖥️ CodeAlpha Python Internship Projects

This repository contains the projects completed as part of the **CodeAlpha Python Programming Internship**.  
The tasks focus on building strong fundamentals in Python including problem-solving, automation, and basic application development.

---

## 🚀 Completed Tasks

### ✅ 1. Hangman Game
A simple text-based game where the player guesses a hidden word letter by letter.

**Features:**
- Predefined word list
- Maximum 6 incorrect attempts
- Displays progress after each guess

**Concepts Used:**
- Loops
- Conditional statements
- Strings & Lists
- Random module

---

### ✅ 2. Stock Portfolio Tracker
A basic Python program to calculate total investment based on user input.

**Features:**
- User inputs stock name and quantity
- Uses predefined stock prices
- Calculates total investment value
- Optional file saving support

**Concepts Used:**
- Dictionary
- Input/Output
- Basic arithmetic
- File handling

---

### ✅ 3. Task Automation Script
Automates simple repetitive tasks using Python.

**Example Implemented:**
- Extract email addresses from a text file and save them

**Concepts Used:**
- Regular Expressions (`re`)
- File handling
- Automation logic

---

### ✅ 4. Basic Chatbot
A simple rule-based chatbot that responds to user inputs.

**Features:**
- Predefined responses
- Continuous interaction loop
- Exit command support

**Concepts Used:**
- Functions
- Conditional statements
- Loops

---

## 🛠️ Technologies Used

- Standard Libraries (random, os, re, etc.)
